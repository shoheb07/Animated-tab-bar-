# Animated-tab-bar-
Creating an animated tab bar in Python is usually done using GUI libraries like Tkinter, PyQt, or Kivy. Below is a clean and beginner-friendly example using Tkinter with a simple animation effect.


## 📌 Description
This project demonstrates how to build a simple **animated tab bar** using Python's built-in **Tkinter** library.  
It features a smooth sliding indicator that moves between tabs when clicked, similar to modern mobile UI designs.


## 🚀 Features
- 🖱️ Clickable tabs (Home, Search, Profile)
- 🎬 Smooth sliding animation for active tab
- 💡 Lightweight and beginner-friendly
- ⚡ No external dependencies required



## 🛠️ Technologies Used
- Python 3.x
- Tkinter (Standard GUI library)


## 📂 Project Structure

animated-tab-bar/ │── app.py          # Main application file │── README.md       # Project documentation
🔧 How It Works
Each tab is a Tkinter Button
An indicator (a small colored bar) highlights the active tab
Animation is achieved using the after() method for smooth transitions
